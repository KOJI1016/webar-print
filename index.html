WebAR Print Kit（アプリ不要）

印刷物と連動するARを、QR→ブラウザだけで体験できるスターターキットです。

① すぐ動く：マーカー式（AR.js / Hiro）

Hiroマーカーを印刷し、スマホで起動した marker.html をかざしてください。

marker.html を開く Hiroマーカーを見る

② 本命：画像トラッキング（MindAR）

  1. 目印にしたい印刷物（ロゴ/ポスター等）の画像を用意
  2. MindARの「Image Target Compiler」で targets.mind を生成
  3. このフォルダに targets.mind を置く
  4. mindar.html を開く → カメラ許可 → 印刷物にかざす

公開のしかた（無料）

  • GitHub Pages / Netlify / Vercel で HTTPS 公開
  • 公開URLをQRコード化 → 生徒配布
